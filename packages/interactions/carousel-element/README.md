# SliderElement

`slider-element` lets you slide child elements

## How to use

### Installation

```sh
npm i --save @webtides/slider-element
```

```js
import '@webtides/slider-element';
```

### Example

```html
<slider-element>
    <li class="item">1</li>
    <li class="item">2</li>
    <li class="item">3</li>
</slider-element>
```
