# SliderElement

`carousel-element` lets you slide child elements

## How to use

### Installation

```sh
npm i --save @webtides/carousel-element
```

```js
import '@webtides/carousel-element';
```

### Example

```html
<carousel-element>
    <div>1</div>
    <div>2</div>
    <div>3</li>
</carousel-element>
```
